# TD Snooker & Pool Tools

Hi! I've been a snooker and billiards enthusiast for years. I'm passionate about the sport, so I've created several apps to make the game easier. I plan to build physical counters for snooker and pool, as well as ball cleaning devices.

## 🌐 Official Website

**Live URL:** [https://dymerskitomasz-del.github.io/Snooker-Pool-Tools/](https://dymerskitomasz-del.github.io/Snooker-Pool-Tools/)

### How to activate GitHub Pages (one-time setup)

1. Go to **Settings → Pages** in this repository.
2. Under **"Build and deployment"**, set **Source** to **`GitHub Actions`** (not "Deploy from a branch").
3. Save. The workflow in `.github/workflows/deploy.yml` handles everything automatically on every push to `main`.

> After merging this PR the workflow runs automatically (~1–2 min) and the site goes live.

## Contents

- **`index.html`** – Main website with Download, Privacy Policy (EN/PL/DE/FR), Upload, and Contact sections.
- **`.github/workflows/deploy.yml`** – GitHub Actions workflow that auto-publishes the site on every push to `main`.

## Uploading APK / EXE files

1. Go to [Releases → New Release](https://github.com/dymerskitomasz-del/Snooker-Pool-Tools/releases/new)
2. Attach your `.apk` or `.exe` files as release assets.
3. Update the download links in `index.html` if needed.
